# Calculator Program (CodeSoft Task 2)

## Description
This is a simple calculator program written in C++.
It performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus

## Concepts Used
- Input/Output
- switch-case
- Operators
- Conditional statements

## How to Run
1. Compile using any C++ compiler
2. Run the program
3. Enter two numbers and an operator
